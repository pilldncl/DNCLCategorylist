## Image Tips

- Use high-quality screenshots taken on an iPhone running iOS 18.6.2 or later
- Ensure text is clearly readable
- Maintain consistent styling/formatting
- Consider dark mode compatibility if needed
- PNG format recommended for screenshots with text
